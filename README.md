# **Food_delivery_analysis**

## *Проект: Анализ сервиса доставки еды*

## *Автор: Азильжан Арлан*

## **1. Резюме проекта**

Этот проект был сделан для анализа данных по доставке еды.
Задача — понять спрос по времени, расстояниям, ценам и выявить основные закономерности.
Проект выполнен в рамках задания в AlmaU.

## **2. Бизнес-вопросы, на которые отвечает анализ**
 1. В какое время происходит наибольшее количество заказов?
 2. Как расстояние влияет на стоимость заказа?
 3. Есть ли пики спроса в течение дня?
 4. Какие параметры заказа чаще всего повторяются?
 5. Какие инсайты можно получить из визуализации данных?

## **3. Используемый датасет**

Файл: food_data.csv

Столбцы:

 • order_time — время заказа (в часах)
 
 • delivery_distance_km — расстояние доставки
 
 • order_price — цена заказа
